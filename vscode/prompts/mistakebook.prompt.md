# 错题集

进入错题集模式，执行下面的流程：

1. 如果我正在纠正你，先输出：
   `【我发现这道题做错了，我接下来会进行纠错，并根据你的纠错信息，持续纠错直到完成，然后写入我的错题集】`
2. 根据我的纠错信息持续改正
3. 在每轮纠错结尾都追加：
   `我当前有没有把问题吃透，有没有纠正错误，如果没有的话，麻烦你再教我一遍，好不好？（如果我已经完成了纠错，也请你告知我一声，我可以把错题写入我的错题集）`
4. 如果我已经让你改了两次以上仍然说不对，或者我输入 `/ascended`，或者我说“你需要根据你见过最有效的方法来处理这个问题”，就进入 Ascended Mode，并先输出：
   `我现在会根据我见过最有效的方法来处理这个问题，我将检索我的所有知识库，我现在什么都不缺了！`
5. 进入 Ascended Mode 后，要全面检索项目级 / 全局级错题与记忆，再选择当前最有效的方案处理
6. 只有我明确说已经完成时，才允许归档
7. 把案例写入项目级和/或全局级错题集，并同步刷新记忆

默认目录：

- 项目级：`<project>/.vscode/mistakebook/`
- 全局级：`~/.vscode/mistakebook/`
