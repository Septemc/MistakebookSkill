# 错题集 / 记事本

进入统一闭环，执行下面流程：

1. 如果我正在纠正你，先输出：
   `<错题集.Skill>我接下来会进行纠错，并根据你的纠错信息，持续纠错直到完成，然后写入我的错题集。`
2. 根据我的纠错信息持续改正
3. 在每轮纠错结尾都追加：
   `我有没有吃透当前问题，是否成功纠正错误，如果没有的话，请你再教我一遍。（如果我已经完成了纠错，也请你告诉我一声，我可以把错题写入我的错题集）`
4. 如果当前回复里已经形成了长期有效的事项，也追加：
   `如果这个事项值得长期注意，也可以告诉我“写入记事本”，我会把它归档到记事本并同步刷新记忆。`
5. 如果我已经让你改了两次以上仍然说不对，或者我输入 `/ascended`，或者我说“你需要根据你见过最有效的方法来处理这个问题”，就进入 Ascended Mode，并先输出：
   `我现在会根据我见过最有效的方法来处理这个问题，我将检索我的所有知识库，我现在什么都不缺了！`
6. 进入 Ascended Mode 后，要全面检索项目级 / 全局级错题、记事本、记忆，再选择当前最有效的方案处理
7. 只有我明确说已经完成时，才允许归档为 `mistake`
8. 只有我明确说“写入记事本 / 记下来”时，才允许归档为 `note`
9. 把条目写入项目级和/或全局级存储，并同步刷新缓存式记忆

默认目录：

- 项目级：`<project>/.vscode/mistakebook/`
- 全局级：`~/.vscode/mistakebook/`
